# chat2api v1.0.0

## Highlights

- Local OpenAI Chat Completions, Anthropic Messages, and Responses-compatible API bridge.
- Expert and vision mode policy, explicit thinking controls, file uploads, SSE, and tool-result continuation.
- Shared .NET Console Host release archives for Windows, Linux, and macOS architectures.
- Context Engine with persisted local conversations, incremental summary support, vector retrieval, and token budgeting.
- Public CI, CodeQL, Dependabot, dependency review, and security reporting configuration.

## Notes

- The default service binding is loopback-only.
- Browser login remains manual and local to the user's environment.
- The Windows tray package has not undergone standalone rigorous testing and is not presented as a separately validated release runtime.
- `fast` is disabled; `expert` is the default and `vision` handles image and file inputs.
- `web_search` is an external tool/MCP contract. chat2api does not execute searches or retain search results.

See the repository README for installation, usage, security guidance, limitations, and the AS IS disclaimer.
